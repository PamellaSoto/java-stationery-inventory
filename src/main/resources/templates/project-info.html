About This Project

Moon Stationery | 月の文房具 (Tsuki no Bunbougu) is a fictional online store designed for fans of anime and aesthetically pleasing stationery. Blending the charm of Japanese pop culture with everyday practicality, this platform offers a seamless shopping experience for customers and an intuitive admin panel for managing products.

Built as a full-stack web application, Moon Stationery showcases officially licensed anime merchandise and cute stationery items in a visually engaging and user-friendly interface.

My Project Goals

  • Create a visually cohesive and accessible storefront
  • Build a robust admin dashboard for easy inventory management
  • Implement intuitive features to enhance the user experience, including search, filters, and product categorization

Tech Stack

  • Backend: Java, Spring Boot, JDBC
  • Frontend: Thymeleaf, HTML, CSS, JavaScript
  • Database: PostgreSQL
  • Architecture: MVC (Model-View-Controller)

Key Features

Storefront

  • Product listings with detailed information on each page
  • Global search bar and dynamic filters by category and anime
  • Breadcrumb navigation for easy browsing
  • Individual product pages with full details and related items

Admin Dashboard

  • Full product management interface
  • Create, edit, or delete products
  • Manage categories
  • Toggle product visibility on the homepage
